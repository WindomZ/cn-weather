# cn-weather-cli

## Usage
```bash
$ cn-weather -h

  Usage: cn-weather [options] [command]


  Commands:

    today <place>  show the weather today in the place

  show chinese weather

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
```
